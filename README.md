# django-social-media-app
A Social Media Web Application built using Django with user authentication, profile management, posts, likes, comments, followers, notifications, and image uploads.
# Social Media Web Application

A full-stack Social Media Web Application built using Django that allows users to create accounts, share posts, interact with other users, and manage their profiles.

## Features

* User Registration and Login
* Secure Authentication
* Create Posts with Images
* Like Posts
* Comment on Posts
* Follow and Unfollow Users
* User Profiles
* Upload Profile Pictures
* Edit Profile Information
* Search Users
* View Followers and Following Lists
* Delete Own Posts and Comments
* Responsive User Interface

## Technologies Used

* Python
* Django
* HTML
* CSS
* Bootstrap
* SQLite
